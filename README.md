# 项目名称： ToDoList
## 项目简介
第一个版本基于原生的JavaScript实现ToDoList的基本功能，如下：
- 添加任务到 **正在进行** 区域
- 单击完成按钮可以完成任务，并添加到 **已经完成** 区域
- 单击删除按钮可以删除任务（从页面和数据库）
- 单击编辑按钮可以修改任务信息
